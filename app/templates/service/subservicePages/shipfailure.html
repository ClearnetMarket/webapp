Answers

What If I don't provide a return method/address?

Back to top
A vendor must provide a means of return within two days of a return request. Failure to do so will auto-grant the BTC back to the customer, and they are no longer obligated by Clearnet Market to return the item.

Its the vendors responsibility to ensure a return address if returns are allowed on the item.

What if the customer doesnt ship the item within 5 days?

Back to top
The customer must mark the item as shipped within 5 days. Failure to do so will auto-grant the BTC to the vendor.

What if I don't receive the item in 14 days?

Back to top
Once the item is marked as sent, if the vendor does not receive the item within 14 days, then the vendor can wait for more time, or mark the item as not returned. The vendor will be auto-granted the BTC.

What happens if I don't mark as received?

Back to top
Once the timer expires, the return will be automatically marked as received. You can extend the time by marking the item as disputed.

What happens if the package is lost/damaged by the courier.

Back to top
It is the shippers responsibility for the delivery of the item. If the vendor shipped, and the item is lost/damaged, the the vendor takes responsibility and can dispute with the carrier. If the customer shipped the item, then the customer takes responsibility, and can dispute any shipping problems with the carrier.

What if we need more time?

Back to top
The customer or vendor can mark the item as disputed. This will make take the item out of the timer.
